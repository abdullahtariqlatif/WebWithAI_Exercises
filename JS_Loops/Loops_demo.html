Interactive guide to JavaScript iteration structures

1. For Loop

The standard for loop is used when you know exactly how many times you want to run a block of code. It consists of three parts: initialization, condition, and increment/decrement.

Logic: 
for (let i = 1; i <= 3; i++) {
  res += `Count: ${i}`;
}

2. While Loop

The while loop loops through a block of code as long as a specified condition is true. Be careful—if the condition never becomes false, you'll create an infinite loop!

Logic: 
let i = 0;
while (i < 3) {
  res += `While: ${i}`;
  i++;
}

3. Do...While Loop

The do...while loop is a variant of the while loop. This loop will execute the code block once before checking if the condition is true.

Logic: 
let i = 5;
do {
  res += `Ran once!`;
  i++;
} while (i < 3);

4. For...of Loop

The for...of statement creates a loop iterating over "iterable" objects like Arrays. It's much cleaner than a standard for loop for accessing items in a list.

Logic: 
const items = ['Apple', 'Orange'];
for (const x of items) {
  res += `Fruit: ${x}`;
}
