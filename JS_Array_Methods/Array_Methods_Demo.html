JS Array Methods Demo

Enter values separated by commas (e.g. 10, 20, apple, 5)

.map() (Multiply)

Multiplies every number by 10.

--

.filter() (Numbers only)

Removes anything that isn't a number.

--

.sort() (Alphabetical)

Default sort (treats items as strings).

--

.sort() (Numeric)

Uses (a, b) => a - b to sort numbers properly.

--

.forEach()

Logs index + value into a string.

--

.concat()

Joins input with ["End"].

--

.every()

Check if ALL items have length > 3.

--

.some()

Check if ANY item is "admin".

--

.includes()

Does the array contain "7"?

--

.join()

Turns array into string with " 🔥 ".

--

.reduce()

Sums numeric values.

--

.find()

Finds first item starting with 'z'.

--

.findIndex()

Index of first item > 50.

--

.indexOf()

Position of "target".

--

.fill()

Fills array with "X".

--

.slice()

Get items from index 1 to 3.

--

.reverse()

Flips the array.

--

.push()

Adds "New" to the end.

--

.pop()

Removes the last item.

--

.shift()

Removes the first item.

--

.unshift()
